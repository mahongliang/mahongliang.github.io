发表于 2017-03-18   |   分类于 生活   |     |   阅读次数

杭州富阳场口科目三必过秘籍

本人2017年3月17号，第二次补考满分飘过，根据前人总结加上自己的经历写出的攻略，每条建议都值500（第二次补考至少需要的花费）啊！！！

预约场次

富阳场口科目三分为1号线、2号线两个考场，单号是1号线，双号是2号线。据网上统计1号线通过率在70%以上，2号线通过率在50%左右。强烈建议选1号线考试，也就是预约单号的时间考试，因为1号线短，路线比较简单，坑较少，一般15分钟就结束，2号线至少20分钟吧，你去实地看一下就知道。

富阳场口科目三分为上午第一场和上午第二场，第一场考完基本上就中午12点了，所以建议选第一场，考完后拿到科目三的成绩单可以去考科目四，然后就可以在当天拿本本了。

排队领号

和科目二一样，科目三也需要线排队领号，虽说是随机排号的，但是建议早点去，我是考了两次都是上午第二场，第一场差不多6点多去的，排到了4700多号，差不多下午5点考的，第二场差不多7点多去的，排到了5700多号，后面没几个人了，差不多下午6点多考的，天都快黑了！！！，感觉他的随机算法是按照时间来的，只是在某一段时间内随机的，先到的比后到的排前面。

扣分项目

科目三考试科目分为：上次准备、模拟灯光考试、直线行驶、超车、加减档、右转、左转、掉头、通过学校、公交站、会车、靠边停车。

中途熄火扣10分；
安全员刹车：扣100分；
起步：不按喇叭 扣5分；不打灯扣 10分；
超车：打转向灯后不到3秒动方向扣 10分；不打灯 10分；
压线：压虚线 扣10分；压实线/双黄线 扣100分；
靠边停车：超过30CM 扣10分；超过50CM扣 100分；
通过路口前未减速慢行扣100分；
遇到情况要中途停车，将车停在路上的交通标志线上，扣100分
调头时，妨碍正常行驶的其他车辆和行人通行，扣100分
指令：听到指令5秒内不操作扣10分。
遇学校区域、人行横道、公交车、会车速度超过30KM/H扣10分；
遇学校区域、人行横道、公交车不摇头观察扣10分；
挂挡：低头看档位扣100分；挂错档位扣10分；
结束：不挂空挡、不拉手刹扣 100分；

1号线攻略

下面我是按照1号线的流程将的，2号线类推

  1. 首先是上车准备，需要检查灯光、安全带、手刹、档位等是否在原位。
  2. 然后开始模拟灯光考试，考完之后关闭灯光。
  3. 起步：分为四步骤，左转向灯、按喇叭、挂一档、放手刹，然后慢慢松离合起步
  4. 起步后有一段直线是供你加速用的，加速到3档，保持25码左右的速度进入直线行驶
  5. 直线行驶：眼睛看远方，保持方向盘不要动就可以了，可以微微的调整方向。
  6. 超车：很简单，注意3秒后动方向就可以了，转回来之后记得要把等关掉！！
  7. 然后过保持20多码左右过弯道
  8. 加减档：踩刹车将速度降低到17码左右，换2档，然后加速到20多码，换3档，然后是一段直线，保持20多码速度，在右转弯处停车
  9. 右转弯：打右转向灯，挂1档起步右转，考场上会有白色线的（教练画的），车头盖住白线转一圈半，然后是一段直线，加速到3档，在下一个右转弯处停车。（红灯时可以右转的）
  10. 右转弯，打右转向灯，挂1档起步右转弯，然后是一段直线，加速到3档，在掉头处停车，
  11. 掉头：打左转灯，挂1档起步掉头，然后是一段直线，加速到3档，在左转弯处停车
  12. 左转弯：打左转向灯，挂1档起步左转弯，然后是一段直线，加速到3档，在下一个左转弯处停车（红灯时不可以左转的）
  13. 左转弯：打左转向灯，挂1档起步左转弯，然后是开回场地内的考道了，这时要经过一个大门，等方向盘回正后，有一个小小的坡道，然后是一段直线，加速到3档，保持20多码的样子
  14. 通过学校：保持20多码的速度，然后左右摇头
  15. 通过公交：保持20多码的速度，然后左右摇头
  16. 会车：保持20多码的速度
  17. 靠边停车：20多码的速度开到靠边停车处停车，然后打右转向灯，挂1档慢慢停车。

以上是1号线的整体考试的整体流程，全程基本上3档20多码在跑，遇到转弯、掉头、靠边停车时停车挂一档在慢慢做动作，打转向灯后3秒钟后动方向盘，起步要缓太离合器，车子动了后要稳牢离合器，等速度指向5码时在放开，这样就完全不会熄火了，科目三其实很简单

上车装备

检查灯光、安全带、手刹、档位等是否在原位。灯光一定要检查远近光灯，是否在原位置上（近光灯的位置），我第一考灯光就直接挂了，车都没开上，就是因为没有检查远近光灯。考试的时候安全员会直接将灯光打在远光的位置，然后模拟灯光考试的时候你就懵逼了。切记切记（第二次补考花了我将近1000，你自己说要不要注意）！！！

起步

分为四步骤，左转向灯、按喇叭、挂一档、放手刹。需要注意的是，打完左转向灯后，一定等三秒钟后，动方向，哪怕方向盘是歪的也不要去修，不然10分就没了。考试车很容易熄火，很多人不过就是因为起步的时候熄火，整场考试需要停车起步的地方有很多，所以起步很关键。教大家一个小技巧，起步要缓太离合器，车子动了后要稳牢离合器，等速度指向5码时再放开，这样就完全不会熄火了。

保持安全距离

在考试的时候，由于考试车辆多，一点要保持较远的距离，宁可停车等待，也不要盲目跟车。个人认为50米吧，这样你才有距离做动作。

停车

这是你最后一步了，在这里挂了确实不应该啊，但是在停车上挂的人也不在少数，所以要特别引起重视。20多码过来，听到请靠边停车指令，打右转灯，踩离合，踩刹车停车后，挂一档起步，在慢慢调整停车位置，车道中间会有一条白虚线，如果车子中线和白虚线重合，那一定没停号，一定会扣分的，白虚线最好跟你重合，在看看反光镜，车子正不正，最后，拉手刹挂空挡，结束考试，

2号线攻略

早上早点起来排队，差不多八点多就刷完卡，大频幕上就会把排的号公布出来，如果是第二场的，建议早上起来不要退房，看到自己的排号就回宾馆睡回笼觉，中午十一二点吃完饭再回来排队把。）

看到大屏幕公布自己的名字后，就去待考区准备待考。进去的时候刷身份证，身份证放上读卡区域3~5秒。就能开门入场，考务员会提示你进去后关闭手机，切记一定要关闭手机。因为规定玩手机发现后，取消考试资格。（一天都等下来了，别因为这个白等了一天，预约多难啊，考试成本多高啊！切记切记！关机！）

在待考区里等待，紧张的小伙伴可以去上个厕所，叫到号后名字在前面的先考试，上车先系好安全带，调整好座椅，后视镜也可以适当的调整一下，下雨的话先擦拭干净！

场内圈操作所有项目
灯光模拟，注意点就是静下心，听完指令后在判断操作。

然后起步，1,2,3,4;1打转左向灯，2打喇叭，3踩离合器挂起步挡，4踩刹车放手制动。做完这些动作后，先松刹车，缓抬离合器，车子开始起步，加油门15码，踩离合换二档，重复加油门25码左右换三档。车子开到主道上，先把车子拉直开平稳，行驶约200米。

开始第一项直线行驶，保持车子直线状态，速度平稳，手握方向盘不要太紧，可以微调方向，大小不超过5度。目视前方300米处作为直线参考。直线开完100米后就结束了。接着准备下一个项目超越前方车辆。

超越前方车辆：行驶约50米；向下打左转向灯，3秒后开始变道，开到左边车道后，关闭左转向灯；约5秒后，驶回原来车道指令出来，向上打右转向灯，3秒后开始变道驶回原来的车道，驶回原车道后关闭右转向灯；这个项目完成。

紧接着就要进入大约300米长的弯道隧道中，注意隧道是有弯度的，方向是往左，车速一定要提前降下来，保持在20码上下过弯道，过了弯道后后是一段直线路程大约300米把，出弯道后加速到25码左右，如果感觉车速够得话就不要加速了，直线过后再又是弯道，注意提前降速，控制在20码左右过弯道。弯道过完准备下一个项目。（过两个弯道前需要减速，控制在20码左右；不放心的话可以体前停车换一档，根据个人习惯，因为停车换挡，怕熄火和没挂进档，而且20码的速度刚好也能平稳通过，第二个弯最好是不要换挡，因为再开50米就加减挡项目了）

加减挡操作：第二个弯道过完后大约50米距离指令加减挡操作，听到后加油门，到17码左右踩离合换2挡，手不要放开操纵杆，脚继续踩住离合器，将档位换回3挡。项目结束，准备下一个项目。（注意点就是换挡前一定要加速，换挡才会顺畅。）

通过学校区域：加减挡操作完成后50米左右，就是学校区域，保持在25码左右的速度，听到指令后左右两边观察一下就完成了。紧接着马上就要出考场大门，准备外圈行驶。

外圈正常驾驶，我会分成好几段给大家介绍，讲的比较详细。

通过学校区域之后大约150米就准备出大门了，指令前方右拐；听到指令向上打右转向灯，停车换一档，准备半联动转弯（跟科目二一样只用离合器前进），缓慢行驶，汽车开到中心线快看不到后，向右打一圈半，车子转到后拐弯车道后迅速回正，保持半联动状态停一下车，右转向灯如果自己灭了的话继续开启，因为出门车子开到看不到黄线又要拐弯一圈半，行驶到公路内车道上（靠近黄线的那个车道），一定不要开错车道，也不要压黄线。

拐到内车道后继续前进，可以加速换二档，前面如果没有车辆的话可以换三档，保持25码上下，行驶到前方T字路口，距离T字路口大约150米，指令提示向左拐弯，向下开启左转向灯，行驶到停车线后，停车换一档，看红绿灯，绿灯情况下半联动状态行驶到前方有标记线（教练偷偷画的，正常情况下是没有的）快看不到的时候右打一圈，刚好拐弯到右边车道。

拐到右边车道后，方向行驶正了后，踩油门加速到15码上下换2挡，继续加速到25码上下换三档，行驶大约300米，指令向右拐弯，听到指令后向上开启右转向灯，开到十字路口后停车换一档，绿灯情况下保持半联动，路口中间那条标记线快看不到后向右打一圈，刚好拐到右边道路内车道里面。开始按照来的线路返回

拐到内车道后车子行驶正了后加油门换挡，开约100米后，指令前方合适路口掉头，开启左转向灯，快到拐弯处停车换1挡，半联动行驶到标记线快看不到向左打一圈，拐到对面方向的内车道里面。车子拐正了之后需要换挡加速，行驶约100米后，指令向左拐弯，向下开启左转向灯。开到路口后停车挂一档，绿灯情况下，行驶到快看不到标记线后半联动打半圈拐弯到左边道路右侧车道，记住不能压黄线。

拐到右侧车道后，加油门挂档行驶200米，指令提示向右拐弯，向上开启右转向灯，行驶至拐弯车道后，停车换一档到T字路口，快要看不到第一条标记线向右打半圈拐弯（如果是第二条标记线向右打一圈拐弯），拐到右边内侧道路，车子拐弯方向正了后加速换挡。

拐弯方向正了后加速换挡。行驶100米左右后，指令提示向左拐弯，向下开启左转向灯，继续行驶100米，红绿灯大门路口，如果是红灯的话一定要停在停车线内，挂一档行驶至标记线快看不见准备向左拐一圈，开进大门后，可以加速换二挡，大概50米左右指令通过公交站区域，车速保持20码上下，左右两边观察一下。加油门换三档,25码左右继续往前行驶约300米后，指令路边停车，踩离合器，刹车降速，停车挂一档，半联动缓慢网路边边线内20公分停靠（如果技术好的，可以不停车换挡，直接三档减速网路边白线20公分距离停车），停车后注意拉手制动，挂空挡，先不要解除安全带。等待考试机器报成绩，听到合格后方可解除安全带，下车后记得礼貌一点和安全员说声谢谢！

总结一下几个注意点把：

  1. 进考场后务必把手机关闭放进口袋或者背包里，不要拿在手里，发现玩手机当场取消考试资格
  2. 上车先系安全带，调整好座椅和后视镜。
  3. 科目三全程车速没超过30码，考试没有限制时间，慢一点稳一点。
  4. 安全员如果催你的话，不要太在意，静下心开自己的车。
  5. 直线行驶可以微调，幅度不要超过5°（方向盘一圈360°，自己把握），微调的时候要提前，发现歪了就开始调整。微调！微调！微调！
  6. 开启转向灯后一定要等3秒后再转弯或者变更车道，行驶到对应车道后及时关闭转向灯。
  7. 过弯道的时候，一定要提前降速到20左右，平稳过弯道。（本人练车测试过，如果是一档的话太慢了，过完弯道加速会比较麻烦，刚过弯道车子还没拉直，突然加速换挡会导致压线，技术好的除外）
  8. 如果前面有车的话一定要保持跟车距离,个人感觉起码要50米开外。
  9. 如果自己很紧张的话可以停车挂一档，仔细想一想平复一下心情。停车不碍事的，不要怕安全员，他们没权利管你停车。
  10. 几乎所有路口准备拐弯的时候都会听见鸣喇叭声，听到后不用紧张，不是后方车辆按的喇叭，是自己考试车上面装的喇叭发出的声音，目的就是要你路口紧张。
  11. 如果车子不小心熄火了，一定要挂空挡踩刹车，再点火发动汽车。
  12. 除了场内内圈拐弯不要换一档，出大门和外圈行驶所有弯道都是挂一档半联动转弯。（类似科目二的速度过弯道。）
